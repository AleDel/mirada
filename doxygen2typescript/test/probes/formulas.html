It represents a 4x4 homogeneous transformation matrix

$T$

[T = \begin{bmatrix} R & t\ 0 & 1\ \end{bmatrix} ]

where

$R$

is a 3x3 rotation matrix and

$t$

is a 3x1 translation vector.

You can specify

$R$

either by a 3x3 rotation matrix or by a 3x1 rotation vector, which is converted to a 3x3 rotation matrix by the Rodrigues formula.

To construct a matrix

$T$

representing first rotation around the axis

$r$

with rotation angle

$|r|$

in radian (right hand rule) and then translation by the vector

$t$

, you can use

  class="normal">cv::Vec3f r, t;


  class="normal">cv::Affine3f T(r, t);

If you already have the rotation matrix

$R$

, then you can use

  class="normal">cv::Matx33f R;


  class="normal">cv::Affine3f T(R, t);

To extract the rotation matrix

$R$

from

$T$

, use

  class="normal">cv::Matx33f R = T.rotation();

To extract the translation vector

$t$

from

$T$

, use

  class="normal">cv::Vec3f t = T.translation();

To extract the rotation vector

$r$

from

$T$

, use

  class="normal">cv::Vec3f r = T.rvec();

Note that since the mapping from rotation vectors to rotation matrices is many to one. The returned rotation vector is not necessarily the one you used before to set the matrix.

If you have two transformations

$T = T_1 * T_2$

, use

  class="normal">cv::Affine3f T, T1, T2;


  class="normal">T = T2.concatenate(T1);

To get the inverse transform of

$T$

, use

  class="normal">cv::Affine3f T, T_inv;


  class="normal">T_inv = T.inv();
